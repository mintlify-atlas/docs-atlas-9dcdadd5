The Plausible Analytics dashboard provides a comprehensive view of your website traffic and visitor behavior. Built with privacy in mind, it delivers powerful insights without compromising user privacy or requiring cookies.

## Overview

Your dashboard displays key metrics and visitor data in an easy-to-understand format. All data updates in real-time, giving you instant visibility into your website's performance.

<Info>
The dashboard is optimized for speed and simplicity. Unlike other analytics platforms, you can see everything you need on a single page without navigating through multiple menus.
</Info>

## Key Metrics

The dashboard tracks essential metrics to help you understand your site's performance:

### Visitor Metrics

- **Visitors** - Unique visitors tracked by user ID (without cookies or persistent identifiers)
- **Visits** - Total number of sessions on your site
- **Pageviews** - Total pages viewed across all visits
- **Views per Visit** - Average number of pages viewed per session
- **Bounce Rate** - Percentage of single-page visits
- **Visit Duration** - Average time visitors spend on your site

<Note>
All metrics are calculated using privacy-friendly methods. Plausible generates a unique user ID from the visitor's IP address, User Agent, and the website domain, which is then hashed and rotated daily.
</Note>

### Additional Metrics

- **Time on Page** - Average time spent on individual pages
- **Exit Rate** - Percentage of visitors who leave from a specific page
- **Scroll Depth** - How far visitors scroll on your pages (when configured)
- **Conversion Rate** - Percentage of visitors completing goals

## Date Range Selection

Customize your view with flexible date ranges:

- **Real-time** - Last 30 minutes
- **Day** - Today's traffic
- **7 days** - Last week
- **30 days** - Last month (default)
- **Month** - Current or previous months
- **Year** - 12-month overview
- **All time** - Complete history
- **Custom** - Select specific date ranges

<Accordion title="How date ranges work">
Date ranges are calculated based on your site's timezone. You can set your preferred timezone in Site Settings. The dashboard automatically adjusts intervals based on the selected range to show the most relevant data granularity.
</Accordion>

## Dashboard Performance

Plausible's dashboard is built for speed:

- **Fast Queries** - ClickHouse database ensures sub-second query times
- **Optimized Loading** - Lazy loading for charts and detailed breakdowns
- **Efficient Caching** - Smart caching reduces server load
- **Responsive Design** - Works seamlessly on desktop and mobile devices

<Warning>
For sites with very high traffic volumes (millions of pageviews), some complex queries may take a few seconds to load. The dashboard uses query optimization to minimize wait times.
</Warning>

## Keyboard Shortcuts

Navigate your dashboard efficiently:

- Press `d` to open date picker
- Press `?` to view all available shortcuts
- Use arrow keys to navigate through data tables
